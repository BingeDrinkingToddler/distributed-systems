Packages required:
Pyro4
xlrd

Start up RMI by running the command python -m Pyro4.naming in console.
Start the system by running each python program.

make requests through client program to the front server and this uses one server to
complete the request and the server used updates the other servers.

not all of the servers are needed for the system to work, if one is started after some
updates it will update itself from the other servers
